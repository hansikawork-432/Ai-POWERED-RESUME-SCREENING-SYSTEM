# AI POWERED RESUME SCREENING SYSTEM
This is a resume screening system which accepts resume in the form of pdf or docx. It therefore converts the pdf into text and collects the data and skills of a person toi match it against the required job description and  further provides a matching score. In this way, it helps recruiters to filter out potential candidates in lesser time as compared to human screening.
